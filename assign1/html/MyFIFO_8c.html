for(int i = 0; i<10;i++)

read_pointer = 0;
write_pointer = 0;
return fifo;

void MyFIFORemove ( int *  )

Removes the oldest element from the FIFO This function uses the read_pointer to find out the oldest element on the FIFO, and removes it from the FIFO. It also checks if the FIFO is empty and if it is, both read_pointer and write_pointer variables return to 0.

fifo[read_pointer] = 0;
read_pointer++;
int check = 0;
for(int i = 0;i<10;i++)
{
if(fifo[i] != 0)
check = 1;
}
if(check == 0)
{
printf("FIFO is empty. Please Add an element before removing\n");
write_pointer = 0;
read_pointer = 0;
}
