◆ StateMachine()

void StateMachine ( )

This function implements the core of the state machine used in the code.
The concept consistes on the states and the transitions that comprise them.
It is also focused on the inner states when a coin it's used. 

- State 0 (Default State)
- State 1 (Add Money)
- State 2 (Return Credit)
- State 3 (Browse Up/Down)
- State 4 (Select Product)
